# 🔍 Searching and Sorting in Java

This repository contains Java implementations of commonly used **Searching** and **Sorting** algorithms. Each program demonstrates the working of an algorithm using simple examples, making it useful for learning Data Structures and Algorithms (DSA).

## 📂 Searching Algorithms
- Linear Search
- Binary Search
  - Rotation Index
  - Peak Element
  - Search in a Sorted 2D Matrix
  - First & Last Occurrence
  - First Missing Positive + Binary Search

## 📂 Sorting Algorithms
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort
- Counting Sort

## 🚀 Technologies Used
- Java
- JDK 8+

## ▶️ How to Run

```bash
javac FileName.java
java FileName
```
